# Check Data Fixer Schema Patch (Java Edition)
In Java Edition, there is a flag which toggles the Check Data Fixer Schema. It is enabled by default. This mod turns it off, allowing you to see one of the game's hidden debugging functions.

**Warning:** Disabling this feature may cause issues. Use at your own risk!

## Installation

To install this mod, place the JAR file in your "mods" directory. You must also install the Ignite mixin loader.

*Note: It is strongly recommended that you use this mod in a separate testing environment.*
